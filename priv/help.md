
The wslogi commands are:
- filter
    - `filter add    <word>`
    - `filter remove <number>`
    - `filter show`
    - `filter ip     [<address>]`
        - default: your ip address  
- help
    - `help`
- level
    - `level list`
    - `level show`
    - `level +<number>`
    - `level -<number>`
    - `level <number> [<number>]`
- start
    - `start`
- stop
    - `stop` 
- trace
    - `trace on`
    - `trace off`
