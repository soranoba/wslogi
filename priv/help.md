
The wslogi commands are:
- filter
    - `filter add    <word>`
    - `filter remove <number>`
    - `filter show`
    - `filter header <key> <value>`
        - Special case:
            - filter header ip [<address>]
                - default: your ip address
                - e.g. {192,168,0,1}
- help
    - `help`
- level
    - `level list`
    - `level show`
    - `level <number>`
    - `level [+ | -]<level> [ [+ | -]<level> ]`
- start
    - `start`
- stop
    - `stop`
